Actividad 1: Elementos de Páginas Web

Tema 1: Elementos de Páginas Web

I. Orígenes de la Web

  1. Web 2.0:
    La Web 2.0 se refiere a la segunda generación de servicios de Internet que permiten a los usuarios colaborar y compartir información en línea de manera interactiva y dinámica. Ejemplos incluyen redes sociales, blogs y wikis.

  2. Web 3.0:
    La Web 3.0, también conocida como la web semántica, es la evolución de la web en la que los datos están interconectados de manera que pueden ser procesados por máquinas. Permite una comprensión más profunda y la ejecución de tareas complejas basadas en los datos disponibles.

  3. La W3C y los estándares:
    El Consorcio World Wide Web (W3C) es una organización que desarrolla estándares web para asegurar la accesibilidad, interoperabilidad y desarrollo a largo plazo de la web. Los estándares incluyen HTML, CSS y XML.

II. Lenguajes de etiquetas

  1. SGML:
    El Lenguaje de Marcado Generalizado Estándar (SGML) es un estándar para definir lenguajes de marcado para documentos. Es la base de lenguajes como HTML y XML.

  2. XML:
    El Lenguaje de Marcado Extensible (XML) es un lenguaje de marcado que define reglas para la codificación de documentos en un formato que puede ser leído tanto por humanos como por máquinas.

  3. HTML:
    El Lenguaje de Marcado de Hipertexto (HTML) es el estándar para crear y diseñar páginas web. Utiliza etiquetas para estructurar el contenido.

  4. XHTML:
    El XHTML es una versión de HTML que sigue las reglas de XML, lo que lo hace más estricto y estructurado.

  5. HTML5:
    HTML5 es la versión más reciente de HTML y agrega nuevas funcionalidades para multimedia y gráficos, mejorando la experiencia del usuario.

III. Otros lenguajes relacionados al desarrollo web

  1. CSS/CSS3:
    Hojas de Estilo en Cascada (CSS) es un lenguaje para estilizar y diseñar el contenido de una página web. CSS3 es la última versión y añade nuevas características como animaciones y transiciones.

  2. JavaScript:
    JavaScript es un lenguaje de programación utilizado para crear contenido interactivo en las páginas web. Permite la creación de dinámicas y experiencias de usuario avanzadas.

  3. Ajax:
    Ajax (Asynchronous JavaScript and XML) es una técnica para crear aplicaciones web interactivas y dinámicas, permitiendo la actualización de contenido sin recargar la página.

  4. Tecnologías y lenguajes del lado del servidor:
    Tecnologías como PHP, Python y Ruby se utilizan para procesar datos en el servidor antes de enviarlos al navegador.

  5. Navegadores:
    Software utilizado para acceder a la web, como Google Chrome, Mozilla Firefox, y Microsoft Edge.

  6. Motores renderizados:
    Componentes de los navegadores que interpretan y muestran el contenido web, como Blink (Chrome) y Gecko (Firefox).

IV. Herramientas de desarrollo

  1. Software WYSIWYG:
    Editores que permiten ver en tiempo real cómo quedará la página web, como Adobe Dreamweaver.

  2. Otras alternativas:
    1. Figma

    2. InVisión

    3. Zeplin

    4. GitHub

  3. Inspección de código desde el navegador:
    Herramienta para inspeccionar y editar HTML, CSS y JavaScript en tiempo real directamente desde el navegador.

Tema 2: El Diseño De Una Página Web

I. Tipos de páginas Web

Existen diferentes tipos de páginas web, cada una diseñada para un propósito específico, como blogs, tiendas en línea, sitios corporativos, portafolios, etc. Cada tipo de página tiene características y funcionalidades únicas que la hacen adecuada para su objetivo.

II. Puntos básicos en la elaboración de una página Web

Consideraciones clave incluyen el diseño visual, la usabilidad, la accesibilidad, la compatibilidad con diferentes dispositivos y navegadores, y la optimización para motores de búsqueda (SEO).

Tema 3: Estructura De Una Página Web

I. Jerarquía de directorio de un proyecto web

La organización y estructura de archivos y directorios en un proyecto web es crucial para la eficiencia y el mantenimiento. Normalmente incluye carpetas para HTML, CSS, JavaScript, imágenes y otros recursos.

II. Elementos de estructura para el diseño de una página Web: Concepto

  1. Regla utilizada para validar y certificar una página web:
    Uso de herramientas como el validador de W3C para asegurar que el código cumple con los estándares.

  2. Reglas para escribir HTML:
    Buenas prácticas incluyen el uso de etiquetas semánticas, anidación correcta y comentarios claros.

  3. Estructura básica de un archivo web:
    <!DOCTYPE html>
    
<html lang="es">
    
<head>
    
    <meta charset="UTF-8">
    
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <title>Título de la Página</title>
    
</head>
    
<body>
    
    <header>...</header>
    
    <nav>...</nav>
    
    <main>...</main>
    
    <footer>...</footer>
    
</body>
    
</html>
